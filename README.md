# 拉屎记录器 (Bowel Movement Recorder)

一个简单易用的排便习惯记录应用，帮助您追踪和分析排便情况。

## 功能特性

- ✅ **添加记录**：快速记录排便时间、地点和备注
- 📋 **历史记录**：查看和管理所有排便记录
- 📊 **数据分析**：
  - 平均间隔时间
  - 本周排便次数
  - 常用地点统计
  - 14天趋势图表
- 🔍 **筛选功能**：按日期范围筛选记录
- 📤 **数据导出**：将记录导出为JSON文件
- 📥 **数据导入**：从JSON文件导入记录
- 💾 **自动备份**：每次修改数据时自动创建备份，防止误操作丢失
- ↩️ **备份恢复**：可查看备份列表，一键恢复到任意历史备份状态
- 🔒 **AES-GCM 加密存储**：记录与设置使用 Web Crypto API 的 AES-256-GCM 加密保存，旧数据向后兼容
- ✅ **表单验证**：校验时间有效性（不能为未来）、地点长度限制
- 👆 **智能滑动手势**：精确区分水平滑动（切换标签）与垂直滚动（浏览内容），支持速度判定与边界阻尼
- ❓ **通用确认模态框**：删除、清空、恢复等危险操作统一使用自定义确认模态框
- 🌙 **深色主题**：在设置中切换深色模式，所有组件均有对应深色样式
- 📱 **移动端优化**：超窄屏适配、刘海屏安全区、按钮最小点击区域、触摸反馈动画
- 🎨 **美观界面**：现代化的设计，流畅的动画效果
- 📱 **响应式设计**：适配各种屏幕尺寸

## 技术栈

- **HTML5**：页面结构
- **Tailwind CSS v3**：样式框架
- **JavaScript (ES6+)**：交互逻辑
- **Chart.js**：数据可视化
- **Font Awesome**：图标库
- **LocalStorage**：本地数据存储
- **Web Crypto API (AES-256-GCM)**：数据加密存储

## 使用方法

### 基本使用

1. **添加记录**：
   - 点击顶部"添加"标签
   - 选择地点（或输入其他地点）
   - 添加备注（可选）
   - 点击"记录"按钮保存

2. **查看历史记录**：
   - 点击顶部"历史"标签
   - 浏览所有记录
   - 点击筛选按钮可按日期范围过滤
   - 点击删除图标可删除记录

3. **查看分析**：
   - 点击顶部"分析"标签
   - 查看平均间隔、本周次数和常用地点
   - 查看14天趋势图表

### 数据管理

- **导出数据**：在历史记录页面点击"导出"按钮
- **导入数据**：在历史记录页面点击"导入"按钮，选择JSON文件
- **创建备份**：在历史记录页面点击"备份"按钮手动创建备份；每次保存/修改记录也会自动创建备份
- **恢复备份**：在历史记录页面点击"恢复"按钮，选择要恢复的备份点
- **切换主题**：在设置中切换浅色/深色模式

## 开发说明

### 本地运行

1. 克隆或下载项目文件
2. 在项目根目录启动本地服务器：
   ```bash
   # 使用Python 3
   python -m http.server 8000
   
   # 或使用Node.js (需要安装http-server)
   npx http-server -p 8000
   ```
3. 在浏览器中访问 `http://localhost:8000`

### 代码修改

- 页面结构：修改 `index.html`
- 样式：修改 `styles.css`
- 逻辑：修改 `script.js`

## 隐私说明

- 所有数据仅存储在您的本地浏览器中
- 不会向任何服务器发送数据
- 记录与设置使用 AES-256-GCM 加密存储于 LocalStorage
- 您可以随时导出数据备份或使用备份/恢复功能
